### Killcount System Overhaul

!!! success "Complete System Rewrite"
    The killcount tracking system has been completely overhauled with session-based tracking and improved visual feedback!

**@killcount Enhanced (Shorthand: @kc)**

**New Features:**
```
- Track specific mob kills in your current session
- Visual killcount display above head upon kill
- Reset function for tracked kills
- "Tracked kills" indicator when using command
- Total killcount preserved across all mobs
- Shorthand @kc command available
```

**Usage:**

| Command | Description |
|---------|-------------|
| `@killcount [MobID]` | Track specific mob |
| `@kc [MobID]` | Shorthand version |
| `@killcount reset` | Reset tracked session kills |

### Instance Improvements

- **Unique ID Lock & Entry Restriction**
  Added Unique ID lock to ET/EC/HTF cooldowns in conjunction with account cooldown. Instances can now only be completed once per character:
  ```
  - Each instance run is tracked by your unique character ID
  - Cannot enter if you're already running the same instance
  - Prevents re-entry after completion
  - Applies across all instance dungeons (ET/EC/HTF)
  - Works in conjunction with account-based cooldowns
  ```

!!! warning "Anti-Circumvention Policy"
    Attempts to circumvent this mechanic will result in all associated account bans.

!!! note "Instance Mechanics"
    This restriction prevents instance farming and ensures balanced progression across all characters.

- **ET & EC Warp Fix**
  Fixed warp out issues in Endless Tower and Endless Cellar instances.

    Complete homunculus storage and switching system implemented! Manage multiple homunculi with ease.
