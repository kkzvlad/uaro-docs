## üêæ Homunculus Changes

!!! info "Homunculus System Rework"
    Major changes to how homunculus spawning and management works!

**Changes:**
```
- No longer auto spawns Homc upon login
- Call Homc reverted to original behavior
- Bioethics skill now an active skill. Used as the interface to swap between
  your stored homc (Cost 1 embryo to swap between)
```

### How It Works

| Action | Description |
|--------|-------------|
| **Call Homc** | Calls your most recently selected homc. Your active choice will not show on your "Bioethics" list of storage |
| **Rest Homc** | Required if you have an active homc before you can access Bioethics to swap |
| **Recall Cost** | Calling your last called homc is still free (No embryo required) |
